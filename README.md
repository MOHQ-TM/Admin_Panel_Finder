# MOHQ Security Team
A tool to find the admin panel.

## Admin Panel Finder
+ A tool to find the admin panel of sites.
+ Written by Python3.
+ have more than 1300 panel.

## Install and run
```bash
git clone https://github.com/MOHQ-TM/Admin_Panel_Finder.git
cd Admin_Panel_Finder
chmod +x install.sh
./install.sh
python3 admin_finder.py
```
